# Brand Style Guide — Bách Hóa Nails

## Màu Sắc Gợi Ý
```
Primary:     #E91E8C (hồng nails — nổi bật, chuyên nghiệp)
Secondary:   #1A1A2E (navy đậm — sang trọng)
Accent:      #FFD700 (vàng — cao cấp)
Background:  #FFF9FC (hồng nhạt gần trắng)
Text:        #1A1A1A (đen)
Text muted:  #666666 (xám)
```

## Typography Gợi Ý
```
Heading: "Playfair Display" hoặc "Cormorant Garamond" (sang trọng)
Body:    "Inter" hoặc "DM Sans" (dễ đọc)
```

## Tone & Voice
- Thân thiện, gần gũi với cộng đồng thợ nails
- Tiếng Việt là ngôn ngữ chính
- Thực tế, hữu ích — không hoa mỹ
- Empowering — "Bạn làm được, tôi chỉ bạn cách"

## Logo Concept
- Tên "Bách Hóa Nails" + icon móng tay/nail polish
- Màu hồng + vàng hoặc hồng + đen

## Đối Tượng Khách Hàng
- Thợ nails người Việt tại Mỹ (chủ yếu phụ nữ 25-50 tuổi)
- Chủ tiệm nails muốn tăng khách
- Dùng điện thoại là chủ yếu → MOBILE FIRST
